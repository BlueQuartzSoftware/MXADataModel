Compile Notes for OS X & Unix Systems
Download, compile and install CMake first. The proceed to build all the third party libraries that are needed. See this page for more information regarding the necessary libraries.

Use CMake to generate Makefiles for your platform. If you are running OS X and you want Xcode projects have CMake geneate those projects for you.

When linking against Boost, prefer to use the static, multi-threaded versions of the boost libraries. There seems to be a bug where linking against the unit_test_framework dynamic library will cause either segmentation faults or the tests will not run properly
